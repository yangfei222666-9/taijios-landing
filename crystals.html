边界

晶体池是模式池 · 不是推荐池 · 请把它当工具 · 不是 oracle

本页仅为模式学习素材 · 不构成金融 / 投资 / 博彩 / 体育竞猜 / 代客决策 / 任何形式的推荐. 依《互联网信息服务管理办法》, 任何基于本页信息的决策风险 100% 由你承担.

非结果承诺. 晶体记录的是历史上"这类卦象模式在这类条件下的命中率", 不是对未来任何单场的承诺. "人越多, 样本验证强度越高" 指的是**样本量对 pattern 本身的校准强度**增加, 不是任何单次结果的准确性保证.

Paper-only 口径. 晶体池本身不对接真钱 API · 不提供投注链接 · 不返佣 · 不荐股. TaijiOS 的 multi_model review spec §12 里 kill switch 明确 "no real-money keys in env".

足球内部化. TaijiOS 的具体足球预测不外发. 晶体池展示的是**模式**而非**具体场次**, 具体比分/赔率在本页面出现时均为**历史已结果**的复盘素材.

护城河的反面. 我们不锁这池子 · 越多人用 · 晶体就越被验证 · 模式就越准. 比起"我的独家数据", 我们要"你和我的共同数据".
